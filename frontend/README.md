# Telegram Bot Admin Panel

React + TypeScript + Vite admin panel for managing Telegram bot tokens and users.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

3. The frontend will run on http://localhost:5173

## Default Login

- Username: `admin`
- Password: `masterkey`

## Build

```bash
npm run build
```

The build output will be in the `dist` directory.
